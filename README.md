# Predicción de Retrasos de Vuelos – Data Science
# Descripción del proyecto

Este repositorio corresponde al área de Data Science del proyecto Predicción de Retrasos de Vuelos. Su objetivo es analizar datos históricos de vuelos para identificar patrones que influyen en la puntualidad y entrenar un modelo predictivo capaz de estimar si un vuelo será puntual o retrasado antes de su despegue.

Los retrasos aéreos generan impactos negativos en pasajeros, aerolíneas y aeropuertos. Mediante el uso de técnicas de ciencia de datos, este proyecto busca anticipar dichos retrasos y apoyar la toma de decisiones operativas.

# Objetivo

Desarrollar un modelo de clasificación binaria que prediga si un vuelo será puntual o retrasado, junto con la probabilidad asociada.

# Alcance

Selección y documentación del dataset.

Exploración y limpieza de datos.

Creación de variables relevantes.

Entrenamiento de un modelo predictivo.

Evaluación del desempeño del modelo.

Exportación del modelo entrenado para su uso en el Back-End.

# Flujo de trabajo

Los datos históricos son explorados y limpiados para garantizar su calidad. Posteriormente, se generan variables que representan características temporales y operativas del vuelo. Con esta información se entrena un modelo supervisado que aprende patrones asociados a los retrasos. Finalmente, el modelo es evaluado y exportado.

# Entregables

Notebook con análisis exploratorio y modelado.

Modelo predictivo serializado.

Documentación del proceso de predicción.

# Tecnologías utilizadas

Python, Pandas, scikit-learn y Jupyter Notebook.
